#### A small soccer game inspired by the classic Amiga game Sensible Soccer.

Requires Python and PyGame.

#### To run:

    python start.py

#### Game controls

    What              Player 1              Player 2 (disabled)
    -----------------------------------------------------------
    Move the player   up/down/left/right    a/s/d/w
    Hard kick         . (period)            ` (backquote)
    Soft kick         / (slash)             1
    -/=               Zoom in and out
