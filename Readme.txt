Добро пожаловать на файловое хранилище студенческих материалов, произведённых по курсу "Операционные системы реального времени"
Данная файловая система должна быть устроена следующим образом:
1) В корневой директории должны находиться папки с номерами групп
2) В папке с номером группы должны находиться папки с фамилиями студентов (если фамилии совпадают, следует добавить имя)
3) В папке с фамилией студента должны находиться папки, по которым можно определить, что за лабораторную работу студент выполнил (с именами lab_1, laboratory_work_1, first_work etc.)
4) Внутри папки с идентификатором лабораторной работы должны содержаться папки с компилируемыми пакетами ROS.

Код предоставляется для проверки по средствам pull request-а.
