Maze Game

Для корректной работы требуются модули:
    - python-xlib
    - pyautogui
    - pynput
    - turtle (вроде стоит автоматом)

Установить можно с помощью команды:
    python -m pip install ModuleName

Для корректной работы графики требуется выделить окно Maze Game сабскрайбера.
В противном случае графен обновляться не будет.

Управление может осуществляться двумя способами:
    - стрелочками (управляет сам сабскрайбер)
    - клавишами w, a, s, d (управляет паблишер)
