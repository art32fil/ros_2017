Запустить rosrun turtlesim turtle_teleop_key /turtle1/cmd_vel:=/RobotControl
Запустить rosrun using_makers shapes

Запустить rviz. Указать Fixed frame - world. Добавить marker. 

Управлять стрелочками в ноде teleop_key.

Роботы состоят из 3 звеньев. Одно основное (главное), вокруг которого вращается второе звено. И третье звено, которое вращается вокург второго.

Условия поражения: столкновение главного звена красного робота с любым из звеньев синего.
Условия победы: нужно поразить третьим звеном главное звено синего робота.
