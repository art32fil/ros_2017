Симулятор состояний двигателей самолета.

(не требуются дополнительные модули)

1) Для корректной работы графики требуется запустить airplane.launch,  в открывшемся окне rviz  переключить Fixed frame на "odom". А так же добавить RobotModel через кнопку add.

2) Для корректной работы приложения необходимо запустить отдельно  repair_engine_server и master, затем запустить roslaunch airplane engines.launch (5 нод двигателей engine1-5).

Ноды будут подовать сигналы о помощи в мастер. Чтобы починить ноду, нужно в отдельном терминале запускать ноду rosrun airplane repair_engine N (где N номер двигателя для починки, пример: rosrun airplane repair_engine 1). По истечению таймеров, ноды уничтожатся. Если их не чинить, то после уничтожения хотя бы 3 нод в мастере отобразится надпись AIRPLANE WAS DESTROYED. И в окне rviz движение модели робота прекратиться.
